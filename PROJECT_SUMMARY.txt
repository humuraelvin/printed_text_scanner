PROJECT OVERVIEW
================================================================================

A production-ready PyQt5 GUI application for optical character recognition (OCR)

Machine Learning by using the Tesseract OCR engine.

Key Technology Stack:

  • OpenCV 4.8.1      - Computer vision and camera handling
  • pytesseract 0.3.10 - Tesseract OCR wrapper
  • Python 3.8+       - Programming language
  • Tesseract 4.0+    - OCR engine (system dependency)

================================================================================
TROUBLESHOOTING GUIDE
================================================================================

Issue: "tesseract is not installed or it's not in your PATH"
Solution: Install Tesseract OCR engine
  $ sudo apt-get install tesseract-ocr    # Linux
  $ brew install tesseract                # macOS
  Download from GitHub (Windows)

Issue: "No module named 'PyQt5'"
Solution: Install PyQt5
  $ pip3 install PyQt5

Issue: "Camera not detected"
Solution:
  1. Check camera connection
  2. Try different camera index (0, 1, 2, etc.)
  3. Linux: Add user to video group
     $ sudo usermod -a -G video $USER
     (Then logout and login)

Issue: "Poor OCR results"
Solution:
  1. Try different preprocessing mode
  2. Adjust threshold value (80-150 usually good)
  3. Ensure good lighting
  4. Use Adaptive Threshold for variable lighting

Issue: "Application window doesn't appear"
Solution:
  1. Check for errors in terminal output
  2. Verify PyQt5 display support
  3. On Linux: Check X11 display settings
  4. Try running with verbose output
